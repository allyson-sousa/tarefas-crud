
# Listar Tarefas

Utilizando o padrão MVC e a biblioteca ArrayList / List, criei um aplicativo simples de lista de tarefas.


## Funcionalidades

- Adicionar Tarefas
- Concluir Tarefas
- Mostrar Tarefas não concluídas
- Mostrar todas as Tarefas
- Sair
